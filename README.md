# CommerceFlow

A production-style e-commerce platform built using Spring Boot Microservices.

## Tech Stack

- Java 21
- Spring Boot
- Spring Cloud
- PostgreSQL
- Kafka
- Redis
- Docker

## Services

- Discovery Server (In Progress)
- Config Server
- API Gateway
- Auth Service
- User Service
- Product Service
- Inventory Service
- Cart Service
- Order Service
- Payment Service
- Notification Service
